# Tools

Small **standalone utilities** — handy helpers that don't warrant a top-level
project of their own. Each one lives in its own folder here with its own README,
so the repo root stays for the bigger projects (SurroundPanner, Galileo Loader).

| Tool | What it does |
|---|---|
| [Working Folders](Working%20Folders/) | One-click Finder-sidebar access to the folders you're working on right now — a small "shelf" of aliases you pin to the sidebar once, instead of fighting Finder tags that Dropbox won't sync. macOS, no installs. |
