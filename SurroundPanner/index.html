Room & speakers ▾

Speakers feed the panner live. Speaker #n drives REAPER output channel n — order matters. X/Y −1…1, Z 0…1; tick LFE for a non‑positional channel.
Coverage (metres) — the area this speaker/zone feeds (0 = off). Ceiling/Sub: ellipse footprint, W/D half-sizes. Wall: a wedge thrown forward — Throw = how far it reaches along Angle, Beam = cone width. Objects outside it get little feed, so the pan follows the real rig.

Panner law ▾

DBAP (distance-based amplitude panning): every object feeds every speaker, weighted by distance, constant-power normalised. Focus tightens each object onto its nearest speakers; Spread widens it across more. The latch lines (links + thicknesses) follow both.
